# 2025-HSG-K9-CPP
Bồi dưỡng HSG Tin học C++ khối 9 – năm học 2024–2025. Bài tập theo tuần, lời giải mẫu, rubric chấm bài.
# KHU VỰC NỘP BÀI CỦA HỌC SINH

⚠️ Học sinh đọc kỹ hướng dẫn này trước khi nộp bài.

---

## 1. Cách tạo thư mục cá nhân

Mỗi học sinh tạo **1 thư mục duy nhất** theo mẫu:

<Lop>_<HoTenKhongDau>

### Ví dụ:
- 9A_NguyenTriLam
- 9B_TranMinhKhoa

❌ Không tạo nhiều thư mục  
❌ Không dùng dấu tiếng Việt  

---

## 2. Cấu trúc nộp bài BẮT BUỘC

Trong thư mục cá nhân, mỗi tuần tạo **1 thư mục riêng**:

A_NguyenTriLam/
├── Tuan01/
├── Tuan02/
├── Tuan03/
└── ...

## 3. Quy định file nộp bài

Trong mỗi thư mục tuần (`Tuan01`, `Tuan02`, …):

### BẮT BUỘC có:
- File bài làm: `.cpp`
- File báo cáo: `report.md`

### Quy ước đặt tên file `.cpp`:
BaiX_<TuanXX>_<HoTenKhongDau>.cpp
VD: Bai1_Tuan01_NguyenTriLam.cpp

---

## 4. Nội dung file report.md (3–5 dòng)

Học sinh ghi ngắn gọn:
- Ý tưởng làm bài
- Cách duyệt mảng / xử lý
- Độ phức tạp (nếu biết)

### Ví dụ:
Em duyệt mảng từ trái sang phải.
Dùng biến dem để đếm số phần tử thỏa điều kiện.
Thuật toán O(n).

---

## 5. LƯU Ý QUAN TRỌNG

❌ Không nộp file .exe  
❌ Không nén .zip / .rar  
❌ Không sửa bài của học sinh khác  
❌ Không nộp sai thư mục tuần  

⚠️ Nộp sai cấu trúc → trừ điểm trình bày

---

## 6. Khi gặp lỗi

- Kiểm tra lại:
  - Tên thư mục
  - Tên file
  - Đúng tuần hay chưa
- Nếu vẫn lỗi, **chụp màn hình + hỏi giáo viên**
